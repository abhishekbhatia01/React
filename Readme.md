# React Projects Repository

A collection of React projects showcasing various concepts, hooks, and techniques.

## Projects Overview

### 1. **react_basic**

Fundamental React concepts and basics.

- Introduction to React components
- State management
- Component lifecycle

### 2. **CustomReact**

Custom React implementation from scratch.

- Understanding React internals
- Virtual DOM concepts
- Core React mechanics

### 3. **react_props_tailwind**

React component development with Tailwind CSS.

- Props drilling and component composition
- Tailwind CSS styling examples
- Card component examples

### 4. **react_hooks**

Deep dive into React Hooks.

- useState, useEffect, and custom hooks
- Hook patterns and best practices
- Examples of hook usage

### 5. **react_project**

General React project template.

- Vite-based setup
- ESLint configuration
- Tailored for quick prototyping

### 6. **currency_converter**

A currency conversion application built with React and Vite.

- Converts between different currencies
- Uses custom hooks for API integration
- Features: InputBox component, currency rate fetching

## Getting Started

1. Navigate to any project directory:

   ```bash
   cd <project-name>
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Tech Stack

- **React** - JavaScript library for building user interfaces
- **Vite** - Next generation frontend tooling
- **Tailwind CSS** - Utility-first CSS framework
- **ESLint** - JavaScript linter
